# Prequisites
* Oh My Zsh
* vim or neovim
* tmux
* Vundle

# Install Instructions
1. `ln ~/.dot-files/.zshrc ~/.zshrc`
1. `ln ~/.dot-files/.vimrc ~/.vimrc`
1. `ln ~/.dot-files/.tmux.conf ~/.tmux.conf`
1. `ln ~/.dot-files/.psqlrc ~/.psqlrc`
1. Launch `vim` and run the vim command `:PluginInstall`
